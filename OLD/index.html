This Project is all about teaching AI how to play a simple game to perfection. In order to train the AI, I used the NEAT algorithm. NEAT is an evolutionary algorithm that creates artificial neural networks. In the end, the AI learns how to play the game perfectly and never dies. I've used the NEAT algorithm on Snake, Chrome Dino Game and Flappy Bird which are all games I built with the Pygame library.

It handles the all the interaction with the user - login, register, contact, verification and everything that is back-end. For the Database I used SQLAlchemy which can easily work with Flask. Home Weather Station Dashboard

This project has a Raspberry Pi - the server, and 5 Arduino's with a temperature sensor. They send the temperature to the Raspberry Pi every 15 minutes and the Raspberry Pi shows it in the localhost webpage with a graph. In addition,the server uses OpenWeatherAPI to get data about the temperature outside such as Humidity, Wind speed, Dew point, etc. It even has a tab for the solar water heating system information.
